void Transceiver::onBitClock ( )
virtual

This method is called in interrupt context

As of September 2020, the digital ramp-down of the Si4463 is broken and not in the latest firmware patch either, so this is a good alternative.

The packet has 2-3 ramp down bits, so when it tells us it's reached them, we ramp the bias voltage down by means of the RC delay circuit. This really helped clean up spurs.
