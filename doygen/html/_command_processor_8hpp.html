Detailed Description

Provides functionality to process incoming commands in the Marine Automatic Identification System firmware.

The CommandProcessor class handles the initialization and processing of commands and events for the transponder firmware. It also includes functionality to switch to bootloader mode, manage One-Time Programmable (OTP) data, and interact with the Command Line Interface (CLI).
