Detailed Description

Manages the active AIS VHF channels in the Marine Automatic Identification System firmware.

This class is responsible for determining and managing the two AIS channels (A and B) that the device uses for communication. It handles the assignment of channels based on events and maintains state information for channel A and channel B.
